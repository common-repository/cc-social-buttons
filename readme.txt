== Description ==

<h3>Description</h3>
All in one contact button with <strong>Premium Contact Buttons</strong> plugin. This plugin allows you to add call, Skype, Facebook, Line, Viber, Zalo, WhatsApp ... anything you want to the website, and give you the best ways to engage your customers

This plugin is totally flexible to change the contact icons or images to make the contact lists fit the style of your website.

<h3>Core features</h3>
<ul>
 	<li>Add a calling button to your phone number, WhatsApp</li>
 	<li>Add a chatting button via Facebook, Line, Viber, Zalo, WhatsApp,...</li>
 	<li>Flexible to change the button icon or image</li>
 	<li>Easy to change to position among buttons.</li>
 	<li>Add as many buttons as you want.</li>
</ul>
